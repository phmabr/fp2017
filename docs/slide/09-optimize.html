Beta Reduction

Beta reduction

The essence of Beta reduction

βε(let x=e1 in e2)={βε,x↦y(e2)βε(e1) is a variable (y)let x=βε(e1) in βε(e2)Otherwise

When a let expression is an alias/renaming (x↦y), the let is removed and occurrences of x in e2 are substituted by y.

Implementation

Beta.g : (Id.t M.t) -> KNormal.t -> KNormal.t

  | Let((x, t), e1, e2) -> (* Beta-reduction of `let` expressions *)
      (match g env e1 with
      | Var(y) ->
          Format.eprintf "beta-reducing %s = %s@." x y;
          g (M.add x y env) e2
      | e1' ->
          let e2' = g env e2 in
          Let((x, t), e1', e2'))

βε(let x=e1 in e2)={βε,x↦y(e2)βε(e1) is a variable (y)let x=βε(e1) in βε(e2)Otherwise

Let Reduction based on Associativity

Let Reduction based on Associativity

The essence of Let Reduction

A(let x=e1 in e2)=let ... in let x=e′1inA(e2)

A(e1) is a nested let form (let ... in e′, where let ...  is a sequence of one or more let’s), and e′1 is not a let form.

Implementation

  | Let(xt, e1, e2) -> insert (f e1)

where

let rec insert = function
| Let(yt, e3, e4) -> Let(yt, e3, insert e4)
| LetRec(fundefs, e) -> LetRec(fundefs, insert e)
| LetTuple(yts, z, e) -> LetTuple(yts, z, insert e)
| e -> Let(xt, e, f e2) in
